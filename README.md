# What is this?

An interactive tool to help manage my Pokemon Tabletop United 1.05 game. This tool will hopefully help both in and out of session.

## Features
* Player Github Page
* DM Tools
* Session Runner

## Tech Stack

* DB: PostgreSQL
* Backend: FastAPI
* ORM: SQLAlchemy
* Schemas: Pydantic
* Migrations: Alembic
* Frontend: Nuxt
